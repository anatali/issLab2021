Getting started with Eventuate Tram

Learn how to get started with Eventuate Tram, a framework for transactional messaging. Eventuate Tram sends and receives messages as part of a database transaction ensuring that your application atomically updates the database and publishes messages. Currently, it supports

  • MySQL and Postgres databases
  • Apache Kafka message broker
